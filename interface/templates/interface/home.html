Help - How to use the interface

The AgroDT Translator Configurator is used to configure protocol translators that convert MQTT, Modbus and MAVLink data into OPC UA.

First, use File → Import AAS... to load an AAS file in .json or .aasx format. After loading, the AAS Tree will appear on the right side of the screen.

Then, click Translators to choose which protocols you want to configure. Each selected translator will appear as a tab in the left panel.

In each translator tab, configure the protocol settings and create mappings between protocol fields and AAS properties. To map a value, select an item in the AAS Tree and bind it to the selected protocol field.

Use Export to generate the configuration files for the selected translators. The status bar at the bottom shows success, warning and error messages while you use the application.
